# Fluent Flow

A voice-powered conversational assistant built with React, TypeScript, and modern web technologies.

## Features

- Speech recognition and synthesis
- Floating interactive orb interface
- Conversation history
- Real-time AI responses
- Responsive design with Tailwind CSS

## Technologies Used

- **Frontend**: React, TypeScript, Vite
- **UI**: shadcn/ui, Tailwind CSS
- **Backend**: Supabase Edge Functions
- **AI**: OpenAI API integration

## Getting Started

### Prerequisites

- Node.js & npm installed
- Supabase account for backend functions

### Installation

1. Clone the repository:
```sh
git clone <YOUR_GIT_URL>
cd fluent-flow
```

2. Install dependencies:
```sh
npm install
```

3. Set up environment variables for Supabase and OpenAI API.

4. Start the development server:
```sh
npm run dev
```

## Deployment

Deploy the frontend to your preferred hosting platform (Vercel, Netlify, etc.) and configure Supabase functions for the backend.
